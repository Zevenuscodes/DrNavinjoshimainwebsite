
Razorpay + WhatsApp + Mongo Booking Add-on

1) Unzip into your project root, merge folders (app/, lib/, models/).
2) npm install
3) Copy .env.example to .env.local and fill keys (Razorpay, Mongo, Twilio).
4) Ensure NEXT_PUBLIC_RAZORPAY_KEY_ID is set.
5) npm run dev
6) Open http://localhost:3000/consultation
